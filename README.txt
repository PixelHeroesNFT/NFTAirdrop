Readme:

Step 1: Install node.js (https://nodejs.org/en/download/)
Step 2: Open the .env file with a text editor (notepad, etc.) and add your 
private key to the private key line
Step 3: Run the .bat files (named 1st step and 2nd step)
one at a time
Step 4: Run the RunAirdrop file
Step 5: Open your browser and put localhost:3000 in the url
and hit enter!

You should be running the airdrop site now!
